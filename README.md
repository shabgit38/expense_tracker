# Expense & Income Tracker

A minimal React + TypeScript expense tracking application with transaction management and monthly summaries.

## Features

✅ **Transaction Management** - Add, edit, delete transactions  
✅ **Monthly Dashboard** - Pivot table summaries by category  
✅ **Sample Data** - Pre-loaded with realistic transactions  
✅ **Type Safe** - Full TypeScript implementation  
✅ **Ready to Extend** - Stubs for charts, forecasting, analytics  

## Quick Start

```bash
npm install
npm run dev
```

Opens at http://localhost:3000

## Project Structure

- `src/components/` - React UI components
- `src/models/` - TypeScript interfaces
- `src/state/` - Context-based state management
- `src/utils/` - Calculation utilities & seed data
